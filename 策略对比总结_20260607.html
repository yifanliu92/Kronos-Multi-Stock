七、Factor Score — 多因子综合评分

定义（Definition）：一个数值（范围 -100 ~ +100），综合多个维度告诉你当前 603305 的多空倾向。
-100 ← 极度看空   |   0 = 中性   |   +100 = 极度看多 →

7.3 盘中的输出形式

在你收到的每10分钟推送里，你会看到这样一行：

[FACTOR_OBSERVER] 因子评分 / Factor Score = +32.5（偏多方向 / Bullish, observer_only=true）
  • +32.5 = 当前综合倾向偏多，但不强烈
  • observer_only=true = 这个数值只供观察，不影响任何交易动作
  • 不影响主策略的 action / reason / position_pct
  • 不影响影子策略的买卖信号
  • 如果因子数据抓取失败，输出 unavailable，不阻断主/影回报

7.4 当前限制（牢记）
Current Limitations

  1. 权重是业务经验设定的，尚未被数据验证。当前值来自"我觉得这个因子应该占多少"，不是"历史数据证明该因子贡献了利润的百分比"。需要积累 A/B 档样本来做回溯验证。
  2. 因子评分不入交易层。未来是否进入下一版策略（如 Candidate-A2），取决于 A/B 档≥20 天后的统计结果。
  3. 小盘股上部分因子效果存疑。例如"开盘偏离度"在 603305 上经常高开低走或低开高走，预测力需要验证。
